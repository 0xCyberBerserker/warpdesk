WarpDesk

A friendlier desktop UI for Cloudflare WARP on Linux, built for people who prefer two clicks over terminal commands for routine tasks.

Made with 🖤 in Barcelona City 🇪🇸

View on GitHub Roadmap

Why it exists

Linux WARP works, but the UX is still too shell-oriented for a small day-to-day task. WarpDesk wraps the official backend in a desktop-first workflow.

What it does

  • Connect and disconnect
  • Switch MASQUE / WireGuard
  • Switch WARP modes
  • Save and apply profiles
  • Inspect diagnostics

Technical stance

WarpDesk does not replace the WARP daemon. It uses warp-cli and warp-svc as the backend and focuses on better Linux desktop ergonomics.
