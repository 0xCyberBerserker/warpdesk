Control

Connecta, desconnecta, reinicia el servei i canvia el mode del túnel des d'una finestra d'escriptori real.

Context

Mantén visibles d'un cop d'ull el diagnòstic, el protocol actiu, els detalls del compte i les accions ràpides.

Flux

Desa perfils repetibles i mantén la interfície alineada amb el tema i l'idioma actius del sistema.

Per què existeix

WARP a Linux funciona. L'experiència d'escriptori necessitava més disciplina.

WarpDesk no intenta substituir el backend oficial. Embolica warp-cli i warp-svc dins d'una GUI més clara i operativa per a qui prefereix context visible abans que memoritzar comandes.

Què inclou avui
  • Estat de connexió i control de protocol i mode
  • Desar, aplicar i esborrar perfils
  • Vista de diagnòstic i dreceres a la safata
  • Suport per a anglès, castellà i català
  • Estil Qt adaptat a la paleta de l'escriptori
Projecte obert

Públic, natiu i deliberadament petit.

El full de ruta viu a GitHub Issues perquè l'empaquetat, el branding, les millores de backend i l'automatització de releases continuïn sent visibles.
